**Create a Hub from the Azure Portal**


Sign in to the Azure Portal.

Search for AI Foundry in the Azure portal.

In the left menu, select AI Hubs.

Click + Create.

Select Hub

Enter:

Hub name

Subscription

Resource Group

Region (Location)
